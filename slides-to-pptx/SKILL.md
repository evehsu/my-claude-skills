# HTML Slides to PPTX Converter

Convert a self-contained HTML slide deck (generated by `/slides`) into a PowerPoint PPTX file by screenshotting each slide and assembling them into a presentation.

## Workflow

### Step 1 — Find the HTML file

If an argument was provided, use it as the input path. Otherwise:
- Search the current working directory for `.html` files: `Glob("**/*.html")`
- Read candidate files briefly to verify they contain `<div class="deck">` and `<section class="slide">`
- If multiple candidates exist or none found, use `AskUserQuestion` to clarify

### Step 2 — Verify it's a slide deck

Read the first ~50 lines of the HTML file and confirm it contains:
- `<div class="deck">`
- At least one `<section class="slide`

If not a slide deck, tell the user and stop.

### Step 3 — Determine output path

Default: same directory and basename as input, with `.pptx` extension.
- Input: `/path/to/my-presentation.html`
- Output: `/path/to/my-presentation.pptx`

If the user specified a different output path, use that instead.

### Step 4 — Run the converter

```bash
python3 slides-to-pptx/convert.py "<input.html>" -o "<output.pptx>"
```

The script will:
1. Auto-install `playwright` if not available (browsers are already cached)
2. Extract speaker notes from `<aside class="notes">` elements
3. Launch headless Chromium at 1280×720 (2× retina) and screenshot each slide
4. Assemble screenshots into a PPTX with full-bleed images and speaker notes

### Step 5 — Report results

After successful conversion, tell the user:
- Number of slides converted
- Full output path
- How to open: `open "<output.pptx>"`

Example: "Converted 14 slides → `/Users/evelyn/my-deck.pptx`. Open with: `open my-deck.pptx`"

## Notes

- **Slides are images** in the PPTX (not editable text) — this preserves pixel-perfect fidelity to the HTML
- **Speaker notes are preserved** as editable text in the PowerPoint notes panel
- **Slide dimensions**: 10" × 5.625" (standard widescreen, matches Google Slides default)
- First run may take ~30s to install playwright; subsequent runs are faster
- Screenshots are taken at 2× device scale factor for crisp retina-quality images
